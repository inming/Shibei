# PDF Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add PDF saving, reading, and annotation support to Shibei, alongside the existing HTML snapshot flow.

**Architecture:** PDF enters as a new `resource_type = "pdf"` value. Storage, sync, search, and annotation tables are unchanged. The frontend conditionally renders a `PDFReader` component (using `pdfjs-dist`) inside `ReaderView` when the resource is a PDF, replacing the iframe. The Chrome extension detects PDF URLs, fetches the raw binary, and POSTs to the existing `/api/save-raw` endpoint with `Content-Type: application/pdf`.

**Tech Stack:** `pdf-extract` (Rust, text extraction), `pdfjs-dist` (npm, rendering + text layer)

**Design Spec:** `docs/superpowers/specs/2026-04-15-v2.3-pdf-support-design.md`

```rust
/// Extract visible text from a PDF file's bytes.
/// Returns empty string on failure (best-effort, mirrors plain_text.rs pattern).
pub fn extract_plain_text(pdf_bytes: &[u8]) -> String {
    match pdf_extract::extract_text_from_mem(pdf_bytes) {
        Ok(text) => {
            // Collapse excessive whitespace, similar to plain_text.rs
            let mut result = String::with_capacity(text.len());
            let mut newline_count = 0;
            for ch in text.chars() {
                if ch == '\n' || ch == '\r' {
                    if ch == '\n' {
                        newline_count += 1;
                        if newline_count <= 2 {
                            result.push('\n');
                        }
                    }
                } else {
                    newline_count = 0;
                    result.push(ch);
                }
            }
            result.trim().to_string()
        }
        Err(_) => String::new(),
    }
}
```

```rust
/// Save a snapshot file with a specified extension (e.g., "html", "pdf").
pub fn save_snapshot_ext(
    base_path: &Path,
    resource_id: &str,
    content: &[u8],
    extension: &str,
) -> Result<PathBuf, StorageError> {
    let dir = resource_dir(base_path, resource_id);
    fs::create_dir_all(&dir)?;
    let filename = format!("snapshot.{}", extension);
    let file_path = dir.join(&filename);
    fs::write(&file_path, content)?;
    Ok(PathBuf::from("storage")
        .join(resource_id)
        .join(filename))
}
```

Keep the existing `save_snapshot()` unchanged — it's used by many callers. New PDF code will call `save_snapshot_ext()`.

```rust
#[tauri::command]
pub async fn cmd_import_pdf(
    state: tauri::State<'_, Arc<AppState>>,
    app: tauri::AppHandle,
    file_path: String,
    folder_id: String,
) -> Result<resources::Resource, CommandError> {
    let source = std::path::PathBuf::from(&file_path);
    let content = std::fs::read(&source).map_err(|e| CommandError {
        message: format!("error.readFileFailed: {}", e),
    })?;

    let resource_id = uuid::Uuid::new_v4().to_string();

    // Save PDF to storage
    let rel_path = storage::save_snapshot_ext(&state.base_dir, &resource_id, &content, "pdf")?;

    // Extract title from filename
    let title = source
        .file_stem()
        .and_then(|s| s.to_str())
        .unwrap_or("Untitled PDF")
        .to_string();

    let conn = state.conn()?;
    let sync_ctx = state.sync_context();

    let resource = resources::create_resource(
        &conn,
        resources::CreateResourceInput {
            id: Some(resource_id.clone()),
            title,
            url: String::new(),
            domain: None,
            author: None,
            description: None,
            folder_id,
            resource_type: "pdf".to_string(),
            file_path: rel_path.to_string_lossy().to_string(),
            captured_at: chrono::Utc::now().to_rfc3339(),
            selection_meta: None,
        },
        sync_ctx.as_ref(),
    )?;

    // Extract plain text (best-effort, in blocking task)
    let content_clone = content.clone();
    let resource_id_clone = resource_id.clone();
    let pool_clone = state.pool.clone();
    tokio::task::spawn_blocking(move || {
        let text = crate::pdf_text::extract_plain_text(&content_clone);
        if !text.is_empty() {
            if let Ok(pool) = pool_clone.read() {
                if let Ok(conn) = pool.get() {
                    let _ = resources::set_plain_text(&conn, &resource_id_clone, &text);
                    let _ = crate::db::search::rebuild_search_index(&conn, &resource_id_clone);
                }
            }
        }
    });

    let _ = app.emit(
        events::DATA_RESOURCE_CHANGED,
        serde_json::json!({
            "action": "created",
            "resource_id": resource.id,
            "folder_id": resource.folder_id,
        }),
    );

    Ok(resource)
}
```

```rust
pub async fn download_snapshot(&self, resource_id: &str, resource_type: &str) -> Result<(), SyncError> {
    let key = Self::snapshot_s3_key(resource_id, resource_type);
    let compressed = self.backend.download(&key).await?;

    let mut decoder = flate2::read::GzDecoder::new(&compressed[..]);
    let mut data = Vec::new();
    std::io::Read::read_to_end(&mut decoder, &mut data)?;

    let ext = if resource_type == "pdf" { "pdf" } else { "html" };
    let local_path = self.base_dir.join("storage").join(resource_id).join(format!("snapshot.{}", ext));
    std::fs::create_dir_all(local_path.parent().unwrap())?;
    std::fs::write(&local_path, &data)?;
    Ok(())
}
```

```tsx
import { useRef, useState, useEffect, useCallback } from "react";
import { useTranslation } from "react-i18next";
import * as pdfjsLib from "pdfjs-dist";
import type { PDFDocumentProxy, PDFPageProxy } from "pdfjs-dist";
import * as cmd from "@/lib/commands";
import type { Highlight, Anchor } from "@/types";
import styles from "./PDFReader.module.css";

pdfjsLib.GlobalWorkerOptions.workerSrc = "/pdf.worker.min.mjs";

interface PDFReaderProps {
  resourceId: string;
  highlights: Highlight[];
  activeHighlightId: string | null;
  onSelection: (info: { text: string; anchor: Anchor; rect: DOMRect }) => void;
  onHighlightClick: (id: string) => void;
  onScroll: (info: { scrollPercent: number; direction: "up" | "down" }) => void;
  onReady: () => void;
}

interface PageInfo {
  pageNum: number;
  width: number;
  height: number;
}

export function PDFReader({
  resourceId,
  highlights,
  activeHighlightId,
  onSelection,
  onHighlightClick,
  onScroll,
  onReady,
}: PDFReaderProps) {
  const { t } = useTranslation("reader");
  const containerRef = useRef<HTMLDivElement>(null);
  const [pdfDoc, setPdfDoc] = useState<PDFDocumentProxy | null>(null);
  const [pages, setPages] = useState<PageInfo[]>([]);
  const [error, setError] = useState<string | null>(null);
  const [loading, setLoading] = useState(true);
  const [needsPassword, setNeedsPassword] = useState(false);
  const [password, setPassword] = useState("");
  const lastScrollY = useRef(0);

  // Track which pages have been rendered
  const renderedPages = useRef<Set<number>>(new Set());
  const canvasRefs = useRef<Map<number, HTMLCanvasElement>>(new Map());
  const textLayerRefs = useRef<Map<number, HTMLDivElement>>(new Map());

  // Load PDF document
  const loadPdf = useCallback(async (pwd?: string) => {
    setLoading(true);
    setError(null);
    try {
      const bytes = await cmd.readPdfBytes(resourceId);
      const data = new Uint8Array(bytes);
      const loadingTask = pdfjsLib.getDocument({
        data,
        password: pwd,
      });
      const doc = await loadingTask.promise;
      setPdfDoc(doc);

      // Gather page dimensions
      const pageInfos: PageInfo[] = [];
      for (let i = 1; i <= doc.numPages; i++) {
        const page = await doc.getPage(i);
        const viewport = page.getViewport({ scale: 1 });
        pageInfos.push({
          pageNum: i,
          width: viewport.width,
          height: viewport.height,
        });
      }
      setPages(pageInfos);
      setNeedsPassword(false);
      onReady();
    } catch (err: unknown) {
      if (err instanceof Error && err.message?.includes("password")) {
        setNeedsPassword(true);
      } else {
        setError(t("pdfLoadError"));
      }
    } finally {
      setLoading(false);
    }
  }, [resourceId, t, onReady]);

  useEffect(() => {
    loadPdf();
    return () => {
      pdfDoc?.destroy();
    };
  }, [resourceId]); // eslint-disable-line react-hooks/exhaustive-deps

  // Render visible pages (virtual scrolling)
  const renderVisiblePages = useCallback(async () => {
    if (!pdfDoc || !containerRef.current) return;

    const container = containerRef.current;
    const scrollTop = container.scrollTop;
    const viewportHeight = container.clientHeight;
    const buffer = viewportHeight; // render ±1 viewport

    // Find which pages are in the visible range
    let accumulatedHeight = 0;
    const containerWidth = container.clientWidth;

    for (const pageInfo of pages) {
      const scale = containerWidth / pageInfo.width;
      const scaledHeight = pageInfo.height * scale;
      const pageTop = accumulatedHeight;
      const pageBottom = accumulatedHeight + scaledHeight;
      accumulatedHeight += scaledHeight + 8; // 8px gap between pages

      const inRange =
        pageBottom >= scrollTop - buffer &&
        pageTop <= scrollTop + viewportHeight + buffer;

      if (inRange && !renderedPages.current.has(pageInfo.pageNum)) {
        renderedPages.current.add(pageInfo.pageNum);
        renderPage(pageInfo.pageNum, scale);
      }
    }
  }, [pdfDoc, pages]);

  const renderPage = async (pageNum: number, scale: number) => {
    if (!pdfDoc) return;
    const page = await pdfDoc.getPage(pageNum);
    const viewport = page.getViewport({ scale });

    // Render canvas
    const canvas = canvasRefs.current.get(pageNum);
    if (canvas) {
      canvas.width = viewport.width;
      canvas.height = viewport.height;
      const ctx = canvas.getContext("2d");
      if (ctx) {
        await page.render({ canvasContext: ctx, viewport }).promise;
      }
    }

    // Render text layer
    const textDiv = textLayerRefs.current.get(pageNum);
    if (textDiv) {
      textDiv.innerHTML = "";
      const textContent = await page.getTextContent();
      pdfjsLib.renderTextLayer({
        textContentSource: textContent,
        container: textDiv,
        viewport,
      });
    }
  };

  // Handle scroll for virtual rendering and progress reporting
  useEffect(() => {
    const container = containerRef.current;
    if (!container) return;

    const handleScroll = () => {
      const scrollY = container.scrollTop;
      const direction = scrollY > lastScrollY.current ? "down" : "up";
      lastScrollY.current = scrollY;

      const maxScroll = container.scrollHeight - container.clientHeight;
      const scrollPercent = maxScroll > 0 ? (scrollY / maxScroll) * 100 : 0;
      onScroll({ scrollPercent, direction });

      renderVisiblePages();
    };

    container.addEventListener("scroll", handleScroll, { passive: true });
    return () => container.removeEventListener("scroll", handleScroll);
  }, [onScroll, renderVisiblePages]);

  // Initial render when pages are ready
  useEffect(() => {
    if (pages.length > 0) {
      renderedPages.current.clear();
      renderVisiblePages();
    }
  }, [pages, renderVisiblePages]);

  // Handle text selection for highlight creation
  useEffect(() => {
    const container = containerRef.current;
    if (!container) return;

    const handleMouseUp = () => {
      const sel = window.getSelection();
      if (!sel || sel.isCollapsed || !sel.rangeCount) return;

      const text = sel.toString().trim();
      if (!text) return;

      // Determine which page the selection is in
      const range = sel.getRangeAt(0);
      const startNode = range.startContainer;
      const pageEl = (startNode instanceof Element ? startNode : startNode.parentElement)?.closest(
        "[data-page-num]"
      );
      if (!pageEl) return;

      const pageNum = parseInt(pageEl.getAttribute("data-page-num") || "0", 10);
      if (!pageNum) return;

      // Calculate charIndex within the page's text layer
      const textLayer = textLayerRefs.current.get(pageNum);
      if (!textLayer) return;

      let charIndex = 0;
      const walker = document.createTreeWalker(textLayer, NodeFilter.SHOW_TEXT);
      let node: Text | null;
      let found = false;
      while ((node = walker.nextNode() as Text | null)) {
        if (node === range.startContainer) {
          charIndex += range.startOffset;
          found = true;
          break;
        }
        charIndex += (node.textContent?.length ?? 0);
      }

      if (!found) return;

      // Build prefix/suffix for fuzzy matching
      const allText = textLayer.textContent || "";
      const prefix = allText.substring(Math.max(0, charIndex - 10), charIndex);
      const suffix = allText.substring(charIndex + text.length, charIndex + text.length + 10);

      const anchor: Anchor = {
        type: "pdf",
        page: pageNum,
        charIndex,
        length: text.length,
        textQuote: { exact: text, prefix, suffix },
      };

      const rect = range.getBoundingClientRect();
      onSelection({ text, anchor, rect });
    };

    container.addEventListener("mouseup", handleMouseUp);
    return () => container.removeEventListener("mouseup", handleMouseUp);
  }, [onSelection]);

  // Password prompt
  if (needsPassword) {
    return (
      <div className={styles.passwordDialog}>
        <p>{t("pdfPasswordRequired")}</p>
        <input
          type="password"
          value={password}
          onChange={(e) => setPassword(e.target.value)}
          placeholder={t("pdfPasswordPlaceholder")}
          onKeyDown={(e) => {
            if (e.key === "Enter") loadPdf(password);
          }}
        />
        <button onClick={() => loadPdf(password)}>{t("pdfUnlock")}</button>
      </div>
    );
  }

  if (loading) {
    return (
      <div className={styles.loading}>
        <div className="spinner" />
        <p>{t("loadingPdf")}</p>
      </div>
    );
  }

  if (error) {
    return (
      <div className={styles.error}>
        <p>{error}</p>
      </div>
    );
  }

  const containerWidth = containerRef.current?.clientWidth ?? 800;

  return (
    <div ref={containerRef} className={styles.container}>
      {pages.map((pageInfo) => {
        const scale = containerWidth / pageInfo.width;
        const scaledWidth = pageInfo.width * scale;
        const scaledHeight = pageInfo.height * scale;

        // Filter highlights for this page
        const pageHighlights = highlights.filter((h) => {
          try {
            const a = JSON.parse(h.anchor);
            return a.type === "pdf" && a.page === pageInfo.pageNum;
          } catch {
            return false;
          }
        });

        return (
          <div
            key={pageInfo.pageNum}
            data-page-num={pageInfo.pageNum}
            className={styles.pageContainer}
            style={{ width: scaledWidth, height: scaledHeight }}
          >
            <canvas
              ref={(el) => {
                if (el) canvasRefs.current.set(pageInfo.pageNum, el);
              }}
            />
            <div
              ref={(el) => {
                if (el) textLayerRefs.current.set(pageInfo.pageNum, el);
              }}
              className={styles.textLayer}
            />
            {/* Render highlights for this page */}
            {pageHighlights.map((h) => {
              // Highlight positioning will be computed in a follow-up render pass
              // after the text layer is populated
              return null; // Placeholder — Task 9 implements highlight rendering
            })}
          </div>
        );
      })}
    </div>
  );
}
```

In `PDFReader.tsx`, add a function to calculate highlight rectangles from the text layer:

```tsx
function getHighlightRects(
  textLayer: HTMLDivElement,
  anchor: { charIndex: number; length: number },
): DOMRect[] {
  const rects: DOMRect[] = [];
  let currentOffset = 0;
  const walker = document.createTreeWalker(textLayer, NodeFilter.SHOW_TEXT);
  let node: Text | null;
  let startNode: Text | null = null;
  let startOffset = 0;
  let endNode: Text | null = null;
  let endOffset = 0;
  const targetStart = anchor.charIndex;
  const targetEnd = anchor.charIndex + anchor.length;

  while ((node = walker.nextNode() as Text | null)) {
    const len = node.textContent?.length ?? 0;
    const nodeEnd = currentOffset + len;

    if (!startNode && nodeEnd > targetStart) {
      startNode = node;
      startOffset = targetStart - currentOffset;
    }
    if (nodeEnd >= targetEnd) {
      endNode = node;
      endOffset = targetEnd - currentOffset;
      break;
    }
    currentOffset = nodeEnd;
  }

  if (startNode && endNode) {
    const range = document.createRange();
    range.setStart(startNode, startOffset);
    range.setEnd(endNode, endOffset);
    const clientRects = range.getClientRects();
    for (let i = 0; i < clientRects.length; i++) {
      rects.push(clientRects[i]);
    }
  }

  return rects;
}
```
