# UX 改进验证清单

## 搜索增强 (Task 1+2)

- [x] 搜索 >= 3 字符的关键词，结果列表中出现匹配上下文片段（snippet，灰色小字，关键词高亮）
- [x] 搜索命中正文时，显示「正文匹配」标签
- [x] 搜索命中标注文本时，显示「标注匹配」标签
- [x] 搜索命中评论时，显示「评论匹配」标签
- [x] 同时命中多个字段时，多个标签并排显示
- [x] 搜索命中标题/URL 时，不显示额外标签（标题本身已高亮）
- [x] 2 字符搜索（LIKE 回退）仍正常工作

## 预览面板概览模式 (Task 3)

- [x] 选中一条资料，右侧预览面板显示：摘要区（description > 正文前200字 > 「暂无摘要」）
- [ ] 显示统计行：`N 条高亮 · M 条评论 · K 条笔记`
- [x] 资料有标签时，显示彩色标签胶囊
- [ ] 底部显示蓝色「在阅读器中打开」按钮，点击正常跳转
- [ ] 不再显示完整的标注列表（与阅读器 AnnotationPanel 去重）

## 阅读器 meta 栏 auto-hide (Task 4)

- [x] 打开阅读器，向下滚动快照内容，meta 栏（标题/URL/日期）向上滑出隐藏
- [x] 向上滚动时，meta 栏重新滑入显示
- [x] 滚动到页面最顶部时，meta 栏始终显示
- [x] 隐藏后 iframe 占满空间，不留空白

## 阅读进度条 (Task 5)

- [x] 阅读器顶部出现 2px 高的蓝色细线
- [x] 滚动快照内容时，进度条宽度跟随滚动百分比变化
- [x] 到达页面底部时进度条 100%
- [x] 页面顶部时进度条接近 0%

## 标注面板折叠 (Task 6)

- [x] 鼠标悬浮在阅读器分割条上，出现 `›` 折叠按钮
- [x] 点击折叠按钮，标注面板收缩为 32px 窄条
- [x] 窄条上显示每个高亮对应颜色的小圆点（竖向排列）
- [x] 窄条底部显示高亮数量
- [x] 点击窄条，标注面板展开回原来宽度
- [x] 折叠/展开后，阅读器 iframe 宽度自适应

## 空状态引导 (Task 7)

- [x] 选中一个空文件夹，显示「没有资料」+ 引导提示（拖拽或插件保存）
- [x] 搜索无结果时，显示「没有搜索结果」+ 引导提示（试试更短关键词）
- [x] 阅读器中快照无标注时，显示引导提示（选中文字后右键）

## 资料列表信息密度 (Task 8)

- [x] 资料项左下角域名前显示标签色点（最多 3 个，对应资料的标签颜色）
- [x] 有标注的资料，右下角日期前显示高亮数量徽章（灰底小数字）
- [x] 无标注的资料不显示数量徽章
- [x] 新增标注后，列表中的计数自动更新

## 回收站增强 (Task 9)

- [x] 打开回收站，顶部显示提示横幅「删除的资料将在 90 天后永久移除」
- [x] 每条删除项显示「剩余 N 天」
- [x] 剩余 <= 7 天的项目，文字显示为红色
- [x] 出现全选 checkbox，勾选后显示「恢复选中 (N)」按钮
- [x] 点击批量恢复，选中项全部恢复

## Sidebar 视觉区分 (Task 10)

- [ ] 文件夹区 section header 右侧显示灰色小字「按位置浏览」
- [ ] 标签区 section header 右侧显示灰色小字「按属性筛选」
- [x] 文件夹区与标签区之间有明显间距
